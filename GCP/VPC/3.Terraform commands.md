# 1.Command to plan terrafrom code using `tfvars`

```
terraform plan -var-file="test.tfvars"
```

# 2.Command to apply terrafrom file using `tfvars`

```
terraform apply -var-file="test.tfvars"
```

```
terraform state list
```

